Get Started

Choose your platform and follow the installation guide

PC (Steam) - Just Dance 2017

1. Installation:

  1. Download the JDParty executable file (click button below).
  2. Locate your Just Dance installation and find the file called 'JD2017.exe'. Replace it with the JDParty executable file you downloaded. (On Steam: Right-click Just Dance 2017 > Manage > Browse Local Files).
  3. Launch the game.

2. Notes:

  • Steam might revert the mod changes if you verify game files or if Ubisoft issues an update (this is rare).
  • This method only works with a legitimate Steam copy of Just Dance 2017.

1. Installation:

  1. Download the JDParty executable file and the uplay_r1_loader.dll (click buttons below).
  2. Locate your Just Dance installation directory. Replace the original 'JD2017.exe' with the JDParty executable you downloaded. Place the 'uplay_r1_loader.dll' in the same directory.
  3. Generate authentication files using the "Generate Auth" button below.
  4. Copy the generated .ini authentication files into your Just Dance 2017 installation directory.
  5. Launch the game using the new JD2017.exe.

2. Notes:

  • This method typically works with specific cracked versions of the game (e.g., by Delusional).
  • Security software (antivirus, firewall, VPN, browser extensions) may flag or block the downloads. These are generally false positives and can be safely ignored if you trust the source.
  • It's highly recommended to back up your original game executable ('JD2017.exe') before replacing it.
  • The authentication files (.ini) are required for the cracked version to connect to JDParty servers.

Nintendo Switch (DNS) - JD19-22

  1. Configure your Nintendo Switch's primary DNS to our custom server address (details provided in the guide).
  2. Launch Just Dance 2019, 2020, 2021, or 2022.
  3. Enjoy JDParty features automatically!

ℹ️ Compatible with Just Dance 2019, 2020, 2021, and 2022 only. No console modification needed.

Nintendo Switch (Patched Game)

  1. Download the pre-patched XCI or NSP game file for your desired Just Dance title.
  2. Install the game file onto your Switch using your preferred Custom Firmware (CFW) installer.
  3. Launch the game. It should auto-connect to JDParty servers.

⚠️ Requires a Nintendo Switch with Custom Firmware (CFW) installed. Understand the risks associated with CFW before proceeding.

Nintendo Switch (DNS via Certificate Bypass - Hacked Switch)

  1. Download the required certificate bypass files.
  2. Place these files onto your Switch's SD card in the specified directory structure (usually related to your CFW's Atmosphere or ReiNX setup).
  3. Ensure your Switch's DNS settings are set to automatic or your standard internet DNS.
  4. Launch Just Dance 2019-2022. The bypass should redirect traffic to JDParty servers.

⚠️ Requires a Nintendo Switch with Custom Firmware (CFW). This method is for Just Dance 2019-2022. Use with caution.
